# Ionic with JWT

<p>Projeto para realizar a autenticação da tela de login usando JWT em uma API.</p>

# Technologies

<p>Ionic 5, JWT, Node, Express, SQLite</p>

## Instalação 
```sh
npm install
```

### Run app
```sh
cd client && ionic serve
```

### Run Server
```sh
cd serve && ionic serve
```
